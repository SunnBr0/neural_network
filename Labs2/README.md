# Нейронная сеть на TensorFlow
Лабораторные работы по работе с python и TensorFlow
## Запуск 
лабораторная с уже готовым набором **mnist** переходим:
```shell
cd exampleFirst
```
После 
```shell
python -u ./main.py
```
лабораторная с набором цифр от 0 до 9 переходим:
```shell
cd second
```
После 
```shell
python -u ./main.py
```
